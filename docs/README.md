# /docs — Hotelly (ADRs + scripts)

**Fonte da Verdade técnica:** `../doc_unificado.md` (Doc Unificado).

Esta pasta mantém apenas:
- `adr/`: registros históricos de decisão (ADRs). Podem estar supersedidos; **não são normativos**.
- `operations/sql/`: scripts operacionais (consulta/limpeza). Executáveis e úteis no dia a dia.
- `ai/schemas/`: schemas versionados usados em contrato/testes.
- `strategy/`: docs de produto/estratégia (não normativos para contratos técnicos).

Qualquer regra de negócio/contrato/invariante deve ser atualizada no Doc Unificado.
